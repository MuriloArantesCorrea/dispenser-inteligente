# 🚿 Dispenser Inteligente

Projeto que simula um dispenser automático com sensor de proximidade e exibe dados em tempo real em uma interface web.

---

## 💡 Funcionalidades

- Detecção de distância (simulada)
- Ativação automática do sistema
- Atualização em tempo real no navegador

---

## 🛠️ Tecnologias utilizadas

- Node.js
- JavaScript
- HTML/CSS

---

## 🚀 Como executar o projeto

1. Instale as dependências:
npm install

2. Inicie o servidor:
node index.js

3. Abra o arquivo index.html no navegador
